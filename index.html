HTML Cheatsheet

Structure Tags

<html> Tag

The <html> element contains the HTML code inside.

<head> Tag

<head> element contains the metadata of the HTML page, like the page title and links to stylesheets.

<title> Tag

<title> element contains the page title, displayed on the browser tab.

<header> Tag

<header> element usually contains the introduction and/or navigation of the page. There can be only one body element on a document.

<body> Tag

<body> element contains the main content of the page. There can be only one body element on a document.

<footer> Tag

<footer> element usually contains information like contact, copyright or links. It can be only one per document.

<section> Tag

<section> groups information in an area

<div> Tag

<div> it's basically a container for block content, grouping in a "box". Usually has a class name to style with CSS

<span> Tag

<span> same idea as <div>, but for inline content.

Text Tags

Heading Tags

Heading can go from 1 to 6, with 1 being the most relevant and 6 the least relevant. Example:

<h1>header</h1>

<h2>header</h2>

<h3>header</h3>

<h4>header</h4>

<h5>header</h5>
<h6>header</h6>

<p> - Paragraph tag

The <p> element is for creating paragraphs

<strong> Emphasis Tag

<strong> represents a content of greater importance, makes the font bold.

<em> - Emphasis Tag

<em> is used to give emphasis to an element, making it italic.

<br> - Breakline tag

<br> is the breakline element, use it to start a
new line. Doesn't need closing tag.

<a> - Anchor tag

<a> defines a hyperlink.

<a href="https://codecademy.com">Codecademy</a>

Codecademy

Anchor tags can also have a property called target=, which when set to "_blank" makes the link open in a new tab.

<a href="https://codecademy.com" target="_blank">Codecademy</a>
