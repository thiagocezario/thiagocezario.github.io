HTML Cheatsheet

Structure Tags

<html>

The <html> element contains the HTML code inside.

<head>

<head> element contains the metadata of the HTML page, like the page title and links to stylesheets.

<title>

<title> element contains the page title, displayed on the browser tab.

<header>

<header> element usually contains the introduction and/or navigation of the page. There can be only one body element on a document.

<body>

<body> element contains the main content of the page. There can be only one body element on a document.

<footer>

<footer> element usually contains information like contact, copyright or links. It can be only one per document.

<section>

<section> groups information in an area

<div>

<div> it's basically a container for block content, grouping in a "box". Usually has a class name to style with CSS

<span>

<span> same idea as <div>, but for inline content.
